Expense Tracker
📌 Overview

The Expense Tracker is a simple application designed to help users record, manage, and analyze their daily expenses. It allows users to track income and spending, categorize transactions, and gain insights into their financial habits for better budgeting and planning.

🚀 Features
Add, update, and delete expenses
Categorize transactions (Food, Travel, Bills, etc.)
Track income and expenses
View transaction history
Generate summaries and reports
User-friendly interface
🛠️ Technologies Used
